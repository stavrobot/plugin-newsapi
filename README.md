# plugin-newsapi

A Stavrobot plugin that provides two tools for searching and browsing news via [NewsAPI.org](https://newsapi.org).

## Getting an API key

1. Go to https://newsapi.org/register and create a free account.
2. Copy the API key from your dashboard.

## Installation

Tell Stavrobot to install `stavrobot/plugin-newsapi`. When prompted, provide your NewsAPI.org API key.

## Tools

- **search_news** — Search articles by keyword, date range, domain, language, and more.
- **top_headlines** — Get current top headlines filtered by country, category, or source.
